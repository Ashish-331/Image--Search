# SnapSeek

SnapSeek is a simple image search web application that allows users to search and view images using a public image API.

## Live Demo
https://ashish-331.github.io/Image--Search/

## Features
- Search images by keyword
- Displays image results dynamically
- Responsive image grid layout
- Clean and minimal UI
- Deployed using GitHub Pages

## Tech Stack
- HTML
- CSS
- JavaScript
- Public Image API
- GitHub Pages (deployment)

## What I Did
- Customized the UI and styling
- Fixed layout and overflow issues using browser DevTools
- Improved usability and visual appearance
- Set up Git workflow and deployed the project
